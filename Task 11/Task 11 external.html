Основи web (HTML, CSS)

Вивчаємо структуру HTML сторінок та взаємодію HTML з CSS

HTML (англ. Hypertext Markup Language) — мова розмітки гіпертексту, що дозволяє за допомогою тегів визначати структуру і зовнішній вигляд HTML-документа (web-сторінки) при відображенні в браузері.Це стандартна мова розмітки для веб-сторінок, яка описує струкутуру сторінки, «пояснює» браузеру, як сторінка має відображатися.

CSS (каскадні таблиці стилів) — мова створення стилів, які визначають зовнішній вигляд елементів на сайті. CSS визначає колір, розмір, шрифт тексту, фон, розташування, вигляд інших елементів сторінки та багато-багато іншого.
